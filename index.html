How to play?

You have a small red robot. It must drill as much oil as possible in the shortest amount of time possible. The robot is controlled by a JavaScript program function that you edit.

Actions

You can only use 1 action per turn (except for robot.useSensor() and robot.getPosition(), which are free).

robot.moveUp()
moves 1 square up
robot.moveRight()
moves 1 square right
robot.moveDown()
moves 1 square down
robot.moveLeft()
moves 1 square left
robot.drill()
drills under the current position
robot.useSensor()
uses the robot's sensors to measure how much oil there is in the surroundings (returns a measurement, see below)
robot.getPosition()
returns the robot's position (see below)

Measurements

The measurements returned from the robot.useSensor() function is an object containing the amount of oil (between 0.0 and 1.0) in the cells adjacent to the robot, as well as the one under it (called center):

{
    center: number,
    top: number | undefined,
    right: number | undefined,
    bottom: number | undefined,
    left: number | undefined,
}

Note that if the robot is adjacent to a wall, that measurement for that cell will be undefined.

Position

The robot's position returned from the robot.getPosition() function is an object of the following format:

{
    x: number,
    y: number,
}
